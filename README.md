#KindEditor富文本编辑器Font-Awesome字体图标版

KindEditor是一款非常强大的Web富文本编辑器，但是其图标样式却是Web1.0时代的产物，和现代浏览器及主流Web页面的设计风格格格不入。  
Font-Awesome是一套非常优雅的Web2.0字体图标库，可以使用CSS的所有特性对它们进行更改，包括：大小、颜色、阴影或者其它任何支持的效果。  

那么问题来了，为什么不将它们结合起来呢？
